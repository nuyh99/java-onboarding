## Problem 1 기능 목록

1. **유효한 페이지 수 인지 검증**
    * 페이지의 범위는 1~400
    * 오른쪽 페이지 = 왼쪽 페이지 + 1
    * 시작면 또는 마지막 면은 제외
    * 왼쪽 페이지는 홀수, 오른쪽 페이지는 짝수
2. **페이지의 각 자리 수를 더하기**
3. **페이지의 각 자리 수를 곱하기**
4. **최종 결과 출력**
   * 포비 승리 : 1
   * 크롱 승리 : 2
   * 무승부 : 0
   * 예외 발생 : -1
